## Misterios del Universo
Es un proyecto web inspirado en el estilo visual y estructural del sitio de Smithsonian. 
    El objetivo principal es presentar información sobre algunos de los fenómenos más fascinantes del universo, utilizando una estructura semántica en HTML5 y estilos personalizados en CSS.
    
Este proyecto fue desarrollado como práctica para reforzar conocimientos en maquetación web, organización de contenido y diseño básico.
Ocupe algunos de mis conocimientos previos de css y alguna que otras funciones de etiquetado HTML para que la página tuviera un poco más de estilo
